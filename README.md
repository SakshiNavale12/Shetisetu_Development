# ShetiSetu - Digital Bridge for Agricultural Governance

A unified digital governance platform connecting Farmers, Agriculture Officers, and Higher Authorities for transparent crop survey, loss verification, and compensation delivery in Maharashtra.

## Project Structure

```
Project A/
├── frontend/          # React + Vite frontend application
│   ├── src/
│   │   ├── components/    # Reusable UI components
│   │   ├── pages/         # Page components
│   │   ├── layouts/       # Role-based layouts
│   │   └── styles/        # Design tokens and styles
│   └── package.json
│
└── backend/           # Node.js backend application
    ├── src/
    └── package.json
```

## Features

### For Farmers (शेतकरी)
- 🌾 Digital crop survey (E-Pik Pahani)
- 📋 Crop loss reporting with GPS-tagged photos
- 📍 Real-time application tracking
- 🔊 Voice assistance in Marathi, Hindi, English
- 📱 Mobile-first, touch-friendly interface

### For Agriculture Officers
- ✅ Digital e-Panchanama creation
- 📸 Field visit photo documentation
- 📊 Task management dashboard
- 🔍 Document verification workflow
- 📈 Performance tracking

### For Higher Authorities
- 📊 Analytics and reporting
- 🗺️ Heat maps and trends
- 👥 Officer performance monitoring
- 📄 Audit and compliance tools
- 💰 Compensation tracking

## Technology Stack

### Frontend
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS 4
- **State Management**: Redux Toolkit
- **Forms**: Formik + Yup
- **Routing**: React Router v7
- **UI Components**: Custom ShetiSetu Design System

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Language**: JavaScript/TypeScript

## Design System

ShetiSetu features a comprehensive design system optimized for:
- Agricultural governance workflows

## Problem Statement

Maharashtra farmers face challenges when crop loss occurs due to natural calamities:
- Information gap on schemes and procedures
- Fragmented workflow across multiple stakeholders
- Lack of standardized digital evidence
- No transparency in application status
- Language and literacy barriers

## Solution

ShetiSetu digitizes the entire workflow from crop survey to compensation delivery with:
- Digital crop survey integration
- GPS-tagged evidence collection
- Real-time status tracking
- Multilingual voice assistance
- Transparent e-Panchanama workflow
